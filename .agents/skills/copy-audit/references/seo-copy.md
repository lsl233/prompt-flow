# SEO Copy Reference

Use this file when reviewing titles, descriptions, headings, intros, anchors, and image text from a search-intent perspective.

## Core Principles

- Match the page copy to the likely query intent.
- Prefer clarity and specificity over cleverness.
- Make the snippet promise consistent with the landing-page content.
- Differentiate similar pages with unique copy.

## HTML Title

Check:

- names the page topic or entity clearly
- includes the differentiating term early
- avoids stuffing repeated keywords
- stays readable when truncated

Flag:

- identical titles across many pages
- titles led by brand only, with the topic hidden late
- vague titles such as "Home", "Products", "Features" without context

## Meta Description

Check:

- previews the real content
- explains value or relevance quickly
- avoids generic filler
- differs across materially different pages

Flag:

- duplicate descriptions
- descriptions that overpromise compared with the page
- descriptions made only of keywords

## H1 and Intro

Check:

- H1 resolves the visitor's immediate question
- intro text explains what the page offers and for whom
- important qualifiers appear early

Flag:

- slogan-only hero copy with no explanatory support
- H1 and title targeting different intents
- intros full of abstract adjectives with little substance

## Image Alt and Title Text

For informative images:

- write alt text based on what the image contributes in context
- keep it concise and specific

For decorative images:

- use empty alt text

Flag:

- alt text copied from filename
- alt text repeating the target keyword unnaturally
- title text used as a hidden SEO field without UX value

## Anchor Text

Check:

- link text makes sense out of context
- adjacent links are distinguishable
- repeated "learn more" or "read more" labels are contextualized nearby

Flag anchor text that gives no hint about destination or task.
