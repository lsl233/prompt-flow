## 四、使用步骤文案

### Step 1
```
中文：保存你的第一个提示词
      在选项页面粘贴或输入提示词，为其添加标签和分类。
      支持纯文本或带变量的模板。

英文：Save Your First Prompt
      Paste or type your prompt in the options page. Add tags and categories.
      Supports plain text or templates with variables.
```

### Step 2
```
中文：在 AI 页面调用
      进入 ChatGPT、Claude 等网站，在输入框中按 ⌘K。
      选择你需要的提示词，自动复制。

英文：Access on AI Pages
      Go to ChatGPT, Claude, or other AI sites. Press ⌘K in the input box.
      Select your prompt, it's automatically copied.
```

### Step 3
```
中文：粘贴并发送
      在 AI 输入框中按 ⌘V 粘贴，直接发送或稍作修改。
      所有提示词本地存储，隐私安全。

英文：Paste and Send
      Press ⌘V to paste in the AI input box. Send directly or edit as needed.
      All prompts stored locally for privacy.
```

## 八、后续优化建议

### Phase 2（上线后）

1. **添加社交证明板块**
   - 用户评价卡片
   - 使用数据统计（"已帮助用户节省 X 小时"）

2. **添加 AI 平台 Logo 墙**
   - 展示支持的 10+ 平台
   - 增强可信度

3. **添加定价板块**
   - 说明免费功能 vs 付费功能
   - 功能点付费说明

4. **添加 FAQ**
   - 常见问题解答
   - 利于 SEO 长尾词

### A/B 测试建议

- 测试不同 Hero 标题的转化率
- 测试 CTA 按钮文案（"免费添加" vs "立即安装"）
- 测试是否有演示视频的影响
